# 花名册（Cat Roster）——玩家说明

快速说明：花名册是你管理、查看与准备猫猫出战/外出的界面。你可以查看每只猫的属性、特质与装备，进行上阵、外出、训练与繁殖等操作。

主要入口与界面
- **花名册（Roster）**：展示你所有的猫卡片，支持筛选（攻击/防御/生命/精力/稀有度）、排序与批量操作。
- **猫详情（Detail）**：查看单只猫的详细信息（属性、技能、父母关系、装备），可以改名、装备或发起训练/繁殖。

玩家能做的事（面向操作）
- **筛选与排序**：快速找出适合当前任务的猫，例如按攻击力降序或按精力筛选可立即外出的猫。
- **上阵/撤下**：把猫放入或移出布阵槽，布阵猫会在战斗或活动中优先被选中。
- **派遣外出（Outing）**：选择一只或多只猫发起外出任务，外出会消耗精力并有机会获得道具、奇物或子代线索。
- **装备奇物**：给猫装备奇物以获得被动效果或战斗增益，某些奇物会影响繁殖/继承概率。
- **改名**：每只猫可**一次性改名**（注意确认）。
- **传习 / 训练**：用经验猫教导其他猫技能，会消耗精力并有成功率/失败惩罚设定。
- **繁殖**：把两只符合条件的猫配对以生成子代，子代继承父母部分属性与特质，稀有子代概率较低。
- **弃置/献祭**：抉择性操作会有二次确认，并可能换取特殊奖励或解锁某些效果。

关键概念（玩家视角）
- **精力**：猫进行外出或训练时消耗的资源，精力为 0 时无法继续行动。
- **奇物**：可装备的特殊道具，能改变战斗或外出结果，也可能影响繁殖继承率。
- **父母/子代关系**：在详情中可查看家谱，方便计划繁殖与培养路线。

新手引导建议
- 第一次进入花名册：引导玩家查看一只示例猫，执行一次改名与一次外出，体验完整流程与结算。
- 强调精力管理：在提示中展示精力恢复机制与可用道具。

界面快捷操作（提升体验）
- **一键筛选**：为常用条件保存快捷筛选（如“战斗队”“外出队”）。
- **批量派遣/回收**：支持将选中猫批量派遣外出或一键回收已完成的外出。
- **快速装备**：在详情中提供“推荐装备”按钮，一键为当前任务套装奇物。

小贴士（玩家向）
- 想要高稀有子代？优先用高稀有父母并在繁殖前给它们装备提升继承率的奇物。
- 遇到高难度外出，优先选择精力充足且拥有适配特质的猫组合。

常见问答（FAQ）
- 问：猫会永久死亡吗？
  - 答：存在死亡机制，但游戏提供保护道具和复活方式，详细说明会在相关界面提示。
- 问：如何快速找到可外出的猫？
  - 答：使用精力筛选或一键“可外出”筛选器，可以立即列出满足条件的猫。
